# Тема курсовой работы: Эксплуатация уязвимостей ARP-протокола при реализации атаки "Человек посередине"

В работе рассматривается разработка и анализ консольного приложения, эксплуатирующего уязвимость ARP протокола с целью проведения атаки «ARP Spoofing». Целью данной атаки является перенаправление трафика двух связанных клиентов одной локальной сети через атакующую машину таким образом, что атакующая машина становится посредником в обмене пакетами данных между клиентами. Приложение, написанное на языке программирования «Python» и использующее библиотеку «Scapy» было протестировано в специально подготовленной локальной сети. Результаты проведенного тестирования доказали работоспособность и эффективность приложения, а также возможность проведения атаки «ARP Spoofing» в ненастроенном специальном образом для противодействия этому сетях. Были сделаны соответствующие выводы о безопасности ARP протокола, а также сформулирован комплексный подход для предотвращения атак, связанных с уязвимостями ARP протокола.
